# Solar Bill OCR Automation

This project extracts important information from electricity bills using OCR and automatically fills data into an Excel file.
# Run ONLY this:
streamlit run sample_bill/app.py
## Features
- Upload electricity bill image
- OCR text extraction
- Extract important bill data
- Save output into Excel
- Download Excel output

## Technologies Used
- Python
- Streamlit
- Tesseract OCR
- OpenPyXL
